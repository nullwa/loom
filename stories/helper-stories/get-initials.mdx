# getInitials

A small utility function to extract initials from a full name.

## Description

The function takes a string containing a full name and returns the initials:

- If the name contains **only one word**, it returns the **first letter**.
- If the name contains **two or more words**, it returns the **first letter of the first two words**.
- Useful for **avatars, user badges, or short labels**.

## Signature

```ts
function getInitials(name: string): string
```

## Parameters

| Name   | Type     | Description                             |
| ------ | -------- | --------------------------------------- |
| `name` | `string` | The full name to extract initials from. |

## Returns

| Type     | Description                              |
| -------- | ---------------------------------------- |
| `string` | Initials derived from the provided name. |

## Usage

```tsx
import { getInitials } from './getInitials'

console.log(getInitials('John Doe')) // "JD"
console.log(getInitials('Alice')) // "A"
console.log(getInitials('Bob Marley')) // "BM"
```

## Examples

#### <b>Single Word Name</b>

```ts
getInitials('Madonna') // "M"
```

#### <b>Two Words</b>

```ts
getInitials('Elon Musk') // "EM"
```

#### <b>Extra Words (only first two count)</b>

```ts
getInitials('Jean Claude Van Damme') // "JC"
```

## Notes

- Trims input automatically when splitting by `" "`.
- Only considers **first two words** even if more exist.
- Case is **not modified**, so it returns the raw first characters.
