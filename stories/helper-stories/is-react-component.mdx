# is-react-component

Utility functions to check if a given value is a valid React component.

It provides specific type guards for **function components**, **class components**, and **forward ref components**.

## API

#### `isFunctionComponent`

```ts
isFunctionComponent(component: any): component is React.FC<any>
```

Checks if the given value is a **function component**.

#### `isClassComponent`

```ts
isClassComponent(component: any): component is React.ComponentClass<any, any>
```

Checks if the given value is a **class component**.

#### `isForwardRefComponent`

```ts
isForwardRefComponent(component: any): component is React.ForwardRefExoticComponent<any>
```

Checks if the given value is a **forward ref component**.

#### `isReactComponent`

```ts
isReactComponent(component: any): component is ReactComponent
```

Checks if the given value is a **valid React component** (function, forward ref, or class).

## Examples

#### <b>Example: Function Component</b>

```tsx
import React from 'react'
import { isFunctionComponent, isReactComponent } from '@/utils/is-react-component'

const MyButton: React.FC<{ label: string }> = ({ label }) => <button>{label}</button>

console.log(isFunctionComponent(MyButton)) // true
console.log(isReactComponent(MyButton)) // true
```

#### <b>Example: Class Component</b>

```tsx
import React from 'react'
import { isClassComponent, isReactComponent } from '@/utils/is-react-component'

class MyInput extends React.Component<{ placeholder: string }> {
  render() {
    return <input placeholder={this.props.placeholder} />
  }
}

console.log(isClassComponent(MyInput)) // true
console.log(isReactComponent(MyInput)) // true
```

#### <b>Example: Forward Ref Component</b>

```tsx
import React, { forwardRef } from 'react'
import { isForwardRefComponent, isReactComponent } from '@/utils/is-react-component'

const MyInput = forwardRef<HTMLInputElement, { placeholder: string }>((props, ref) => <input ref={ref} {...props} />)

console.log(isForwardRefComponent(MyInput)) // true
console.log(isReactComponent(MyInput)) // true
```

## Notes

- `any` is used intentionally instead of `unknown` because type narrowing would break the type assertions in `isReactComponent`.
- This is especially useful when working with **dynamic rendering** where components may be passed as props, or when validating a component before rendering.
