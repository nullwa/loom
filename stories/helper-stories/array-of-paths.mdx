# get-breadcrumb-items

Utility function to **generate breadcrumb navigation items** from a pathname.

It splits a URL path into segments, creates human-readable labels, and accumulates their corresponding href values.

## API

#### `getBreadcrumbItems`

```ts
getBreadcrumbItems(pathname: string): { label: ReactNode; href: string }[]
```

- **`pathname`**: A string representing the current route (e.g. `"/dashboard/settings"`).
- **Returns**: An array of breadcrumb items with:

  - `label`: A `ReactNode` (prettified label).
  - `href`: The accumulated path segment.

## Examples

#### <b>Example: Root Path</b>

```tsx
import { getBreadcrumbItems } from '@/utils/get-breadcrumb-items'

console.log(getBreadcrumbItems('/'))
// → [{ label: 'Home', href: '/' }]
```

#### <b>Example: Simple Path</b>

```tsx
import { getBreadcrumbItems } from '@/utils/get-breadcrumb-items'

console.log(getBreadcrumbItems('/dashboard'))
// → [
//     { label: 'Home', href: '/' },
//     { label: 'Dashboard', href: '/dashboard' }
//   ]
```

#### <b>Example: Nested Path</b>

```tsx
import { getBreadcrumbItems } from '@/utils/get-breadcrumb-items'

console.log(getBreadcrumbItems('/dashboard/user-settings/profile'))
// → [
//     { label: 'Home', href: '/' },
//     { label: 'Dashboard', href: '/dashboard' },
//     { label: 'User Settings', href: '/dashboard/user-settings' },
//     { label: 'Profile', href: '/dashboard/user-settings/profile' }
//   ]
```

## Notes

- **Prettification**:

  - Hyphens (`-`) are replaced with spaces.
  - First letters are capitalized (e.g. `"user-settings"` → `"User Settings"`).

- Always starts with `{ label: "Home", href: "/" }`.

## Usage with a Breadcrumb Component

```tsx
import Link from 'next/link'
import { getBreadcrumbItems } from '@/utils/get-breadcrumb-items'

const Breadcrumb = ({ pathname }: { pathname: string }) => {
  const items = getBreadcrumbItems(pathname)

  return (
    <nav>
      {items.map((item, idx) => (
        <span key={item.href}>
          <Link href={item.href}>{item.label}</Link>
          {idx < items.length - 1 && ' / '}
        </span>
      ))}
    </nav>
  )
}
```

✅ This makes it easy to **render breadcrumbs dynamically** based on the current route in **Next.js or React apps**.
