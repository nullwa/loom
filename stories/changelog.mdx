# Changelog

All notable changes to **Loom ui** are tracked in a **semantic, API-style changelog**, following a simple workflow:

- **feat** – A new feature
- **fix / bugfix** – A bug fix
- **chore** – Changes to build, docs, or non-user-facing code

<br />
<br />
